# H2F Gallery

Allows embedding a clean, organized, and interactive Card Gallery representation of a Google Sheets table into our Google Website. 

#### Features

 ###### Embedding, Searching, filtering, pagination, & mobile friendly UI/UX.

## Sources

Hosting + Code Repo: Gitub

"Database": Google Sheets + Visualization API

Codespace + IDE: Firebase Studio

Front + Backend: Vite + Preact + Typescript

## Usage

For our [H2F Webpage](https://sites.google.com/view/mnh2f/home).
